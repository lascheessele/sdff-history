# SDFF League History

Static fantasy-football league dashboard prepared for GitHub Pages.

## Publish with GitHub Pages

1. Create a new GitHub repository, for example `sdff-history`.
2. Upload `index.html` and `.nojekyll` to the repository root.
3. Open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder, then click **Save**.
6. GitHub will provide a public URL, typically:
   `https://YOUR-USERNAME.github.io/sdff-history/`

The dashboard is fully static and does not contain ESPN login cookies.
